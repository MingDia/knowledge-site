# PARA自动整理工作流

## 📋 工作流概述
本工作流实现从Inbox暂存箱到PARA分类体系的自动化整理，帮助你快速处理未分类笔记。

## 🔍 分类规则
### 1. Projects（项目）
**判定条件**：
- 包含明确的项目名称和截止时间
- 有具体的待办任务列表
- 属于当前正在进行的、有明确完成目标的工作
- 标签包含 `#project` 或 `#项目`

**示例**：
- 2026年Q2产品迭代计划.md
- 客户A项目需求文档.md

### 2. Areas（领域）
**判定条件**：
- 属于长期关注的领域/主题
- 需要持续维护和更新的内容
- 与个人职责、兴趣领域相关
- 标签包含 `#area` 或 `#领域`

**示例**：
- 编程技能学习笔记.md
- 个人健康管理.md
- 财务管理记录.md

### 3. Resources（资源）
**判定条件**：
- 参考资料、知识库内容
- 收集的各类文章、教程、素材
- 没有明确的时间限制，供随时查阅
- 标签包含 `#resource` 或 `#资源`

**示例**：
- Python最佳实践指南.md
- 设计灵感收集.md
- 常用工具清单.md

### 4. Archives（归档）
**判定条件**：
- 已完成的项目文档
- 不再活跃的领域内容
- 过时的参考资料
- 标签包含 `#archive` 或 `#归档`

## 🚀 使用方法
1. **收集阶段**：所有新创建的笔记、临时记录、未分类内容全部先存入 `Inbox` 文件夹
2. **标记阶段**：给Inbox中的笔记添加对应的标签（`#project`/`#area`/`#resource`/`#archive`）或者在frontmatter中添加`category`属性
3. **公开标记**：对于可以公开发布的笔记，在 frontmatter 中添加 `public: true`
   - 检查内容：确保不包含私密信息（姓名、邮箱、财务数据等）
   - 必需字段：同时添加 `title` 和 `description` 字段
4. **整理阶段**：发送「整理PARA」命令，系统会自动：
   - 扫描Inbox文件夹下的所有文件
   - 根据标签或frontmatter属性自动分类到对应的PARA文件夹
   - 无法自动识别的文件会保留在Inbox并提示人工处理
   - 自动更新所有内部链接，确保链接有效性
   - 对于无法自动识别的文件，会提示人工处理

## 📝 手动整理提示
对于无法自动识别的文件，你可以：
1. 手动移动到对应的PARA文件夹
2. 添加正确的标签后再次运行整理命令
3. 对于不确定分类的内容可以继续暂存在Inbox，后续再处理

## ⚙️ 整理命令说明
- **命令**：`整理PARA` 或 `整理暂存箱`
- **效果**：自动整理Inbox内所有可识别的笔记到对应分类
- **输出**：整理完成后会输出整理报告，包含移动的文件列表和待人工处理的文件列表

:::tip 💡 建议
每周固定时间运行一次整理命令，保持知识库的整洁有序。
:::
